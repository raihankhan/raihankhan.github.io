Linux File Systems

Directory Structure The directories help us to store the files and locate them when we need them. Also, directories are called folders as they can be assumed of as folders where files reside in the form of a physical desktop analogy. Directories can be organized in a tree-like hierarchy in Linux and several other operating systems. The directory structure of Linux is well-documented and defined in the Linux FHS (Filesystem Hierarchy Standard).
